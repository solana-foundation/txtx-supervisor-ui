## Getting started

Install `txtx`:
```console
curl -sL https://install.txtx.sh/ | bash
```

Start a runbook
```console
# Clone BNS repo
git clone git@github.com:txtx/BNS-V2.git

# Change directory
cd BNS-V2

# Move to branch
git checkout feat/txtx-runbooks

# List all the runbooks
txtx ls

# Execute a runbook
txtx run register-bns-name
```

Clone this repo:
```console
git clone git@github.com:txtx/txtx-web-ui.git

cd txtx-web-ui

npm install

npm run dev
```

Open Chrome on `http://localhost:1234`
